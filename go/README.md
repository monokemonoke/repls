# REPL in Go

このフォルダは, Go での REPL の実装を行ったものになります.

## 動作確認環境

```
$ sw_vers
ProductName:	macOS
ProductVersion:	12.6
BuildVersion:	21G115

$ go version
go version go1.16.4 darwin/amd64
```

## How To Run

1. 以下のコマンドで REPL が立ち上がります.

```
$ go run main.go
```
