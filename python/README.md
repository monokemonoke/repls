# REPL in Python

このフォルダは, Python での REPL の実装を行ったものになります.

## 動作確認環境

```
$ sw_vers
ProductName:	macOS
ProductVersion:	12.6
BuildVersion:	21G115

$ python3 --version
Python 3.10.6
```

## How To Run

1. 以下のコマンドで REPL が立ち上がります.

```
$ python3.10 -m venv .venv
$ source .venv/bin/activate'
$ python3 main.py
```
