# REPL in Ruby 

このフォルダは, Ruby での REPL の実装を行ったものになります.

## 動作確認環境

```
$ sw_vers
ProductName:	macOS
ProductVersion:	12.6
BuildVersion:	21G115

$ ruby --version
ruby 2.6.8p205 (2021-07-07 revision 67951) [universal.x86_64-darwin21]
```

## How To Run

1. 以下のコマンドで REPL が立ち上がります.

```
$ ruby main.rb
```
